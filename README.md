# Healthcare Appointment Analysis

## Project Overview
This project analyzes healthcare appointment data to identify patterns related to patient no-shows, waiting time, gender distribution, and appointment trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- SQL
- Power BI

## Objectives
- Analyze patient no-show behavior
- Study appointment trends
- Improve hospital efficiency
- Create interactive dashboards

## Dataset
Kaggle Medical Appointment Dataset

## Features
- Data Cleaning
- Exploratory Data Analysis
- SQL KPI Queries
- Power BI Dashboard
- Business Insights

## KPIs
- No-show Rate
- Appointment Completion Rate
- Gender-wise Attendance
- Age-wise Analysis
- SMS Reminder Effectiveness

## Dashboard Pages
1. Executive Dashboard
2. Appointment Analysis
3. Patient Demographics
4. No-show Analysis
5. Operational Insights
